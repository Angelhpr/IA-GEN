# Práctica integradora de variables y tipos

## Propósito

Esta práctica reúne los conocimientos estudiados durante el módulo:

- Variables.
- Asignación y reasignación.
- Tipos de datos básicos.
- Entrada de datos con `input()`.
- Conversiones con `int()`, `float()` y `str()`.
- Operadores aritméticos.
- Precedencia de operadores.
- Asignación aumentada.
- Uso de `type()`.

El objetivo es construir programas pequeños que reciban información, realicen cálculos y muestren resultados claros.

## Recomendaciones

Antes de comenzar:

1. Crea una carpeta para guardar los ejercicios.
2. Utiliza nombres descriptivos para los archivos.
3. Ejecuta cada programa después de escribirlo.
4. Lee cuidadosamente cualquier mensaje de error.
5. Comprueba los tipos de datos cuando sea necesario.
6. Utiliza nombres de variables en `snake_case`.
7. Guarda los archivos con extensión `.py`.

## Actividad 1: ficha de estudiante

Crea un archivo llamado:

~~~text
ficha_estudiante.py
~~~

El programa debe almacenar:

- Nombre del estudiante.
- Edad.
- País.
- Nombre del curso.
- Número del módulo.
- Promedio.
- Estado del curso.
- Calificación final todavía no disponible.

Utiliza los tipos de datos correctos.

Ejemplo:

~~~python
nombre_estudiante = "Ana"
edad = 20
pais = "Honduras"
nombre_curso = "Python desde cero"
numero_modulo = 2
promedio = 91.5
curso_activo = True
calificacion_final = None

print("Nombre:", nombre_estudiante)
print("Edad:", edad)
print("País:", pais)
print("Curso:", nombre_curso)
print("Módulo:", numero_modulo)
print("Promedio:", promedio)
print("Curso activo:", curso_activo)
print("Calificación final:", calificacion_final)
~~~

Después muestra el tipo de cada variable mediante `type()`.

## Actividad 2: actualización de progreso

Crea un archivo llamado:

~~~text
actualizar_progreso.py
~~~

Inicia con estos valores:

~~~python
ejercicios_completados = 8
lecciones_completadas = 3
puntos = 120
~~~

Realiza las siguientes actualizaciones:

- Aumenta los ejercicios completados en `2`.
- Aumenta las lecciones completadas en `1`.
- Aumenta los puntos en `50`.
- Multiplica los puntos por `2`.
- Resta `20` puntos.

Utiliza operadores de asignación aumentada.

Ejemplo:

~~~python
ejercicios_completados = 8
lecciones_completadas = 3
puntos = 120

ejercicios_completados += 2
lecciones_completadas += 1
puntos += 50
puntos *= 2
puntos -= 20

print("Ejercicios completados:", ejercicios_completados)
print("Lecciones completadas:", lecciones_completadas)
print("Puntos:", puntos)
~~~

Resultado esperado:

~~~text
Ejercicios completados: 10
Lecciones completadas: 4
Puntos: 320
~~~

## Actividad 4: calculadora de progreso

Crea un archivo llamado:

~~~text
calculadora_progreso.py
~~~

El programa debe solicitar:

- Cantidad total de lecciones.
- Cantidad de lecciones completadas.
- Cantidad de ejercicios por lección.

Debe calcular:

- Lecciones pendientes.
- Total estimado de ejercicios.
- Ejercicios correspondientes a las lecciones completadas.

Ejemplo:

~~~python
total_lecciones = int(
    input("Cantidad total de lecciones: ")
)
lecciones_completadas = int(
    input("Lecciones completadas: ")
)
ejercicios_por_leccion = int(
    input("Ejercicios por lección: ")
)

lecciones_pendientes = (
    total_lecciones - lecciones_completadas
)

total_ejercicios = (
    total_lecciones * ejercicios_por_leccion
)

ejercicios_completados = (
    lecciones_completadas * ejercicios_por_leccion
)

print()
print("Lecciones pendientes:", lecciones_pendientes)
print("Total de ejercicios:", total_ejercicios)
print(
    "Ejercicios completados:",
    ejercicios_completados
)
~~~

## Actividad 6: división y residuo

Crea un archivo llamado:

~~~text
grupos_estudiantes.py
~~~

El programa debe solicitar:

- Cantidad total de estudiantes.
- Cantidad de estudiantes por grupo.

Debe calcular:

- Cantidad de grupos completos.
- Cantidad de estudiantes restantes.

Ejemplo:

~~~python
total_estudiantes = int(
    input("Total de estudiantes: ")
)
estudiantes_por_grupo = int(
    input("Estudiantes por grupo: ")
)

grupos_completos = (
    total_estudiantes // estudiantes_por_grupo
)

estudiantes_restantes = (
    total_estudiantes % estudiantes_por_grupo
)

print("Grupos completos:", grupos_completos)
print(
    "Estudiantes restantes:",
    estudiantes_restantes
)
~~~

Utiliza una cantidad por grupo diferente de cero.

## Proyecto integrador: resumen de inscripción

Crea un archivo llamado:

~~~text
resumen_inscripcion.py
~~~

El programa debe solicitar:

- Nombre del estudiante.
- Edad.
- Nombre del curso.
- Cantidad de módulos.
- Precio por módulo.
- Porcentaje de descuento.
- Cantidad de pagos.

Debe calcular:

- Edad del próximo año.
- Precio sin descuento.
- Valor del descuento.
- Precio final.
- Valor de cada pago.
- Cantidad de módulos por pago.
- Módulos restantes.

Utiliza este código como referencia:

~~~python
nombre = input("Nombre del estudiante: ")
edad = int(input("Edad: "))
curso = input("Nombre del curso: ")
cantidad_modulos = int(
    input("Cantidad de módulos: ")
)
precio_modulo = float(
    input("Precio por módulo: ")
)
descuento_porcentaje = float(
    input("Porcentaje de descuento: ")
)
cantidad_pagos = int(
    input("Cantidad de pagos: ")
)

edad_proxima = edad + 1

precio_sin_descuento = (
    cantidad_modulos * precio_modulo
)

valor_descuento = (
    precio_sin_descuento
    * descuento_porcentaje
    / 100
)

precio_final = (
    precio_sin_descuento - valor_descuento
)

valor_pago = precio_final / cantidad_pagos

modulos_por_pago = (
    cantidad_modulos // cantidad_pagos
)

modulos_restantes = (
    cantidad_modulos % cantidad_pagos
)

print()
print("Resumen de inscripción")
print("Estudiante:", nombre)
print("Edad:", edad)
print("Edad el próximo año:", edad_proxima)
print("Curso:", curso)
print("Cantidad de módulos:", cantidad_modulos)
print("Precio sin descuento:", precio_sin_descuento)
print("Descuento:", valor_descuento)
print("Precio final:", precio_final)
print("Valor por pago:", valor_pago)
print("Módulos por pago:", modulos_por_pago)
print("Módulos restantes:", modulos_restantes)
~~~

Para realizar la práctica:

- Utiliza una cantidad de pagos diferente de cero.
- Comprueba los resultados manualmente.
- Consulta los tipos de las variables principales.
- Cambia los datos y ejecuta nuevamente el programa.
- Observa qué resultados se modifican.
